## Brand platform

vanityURLs is a practical redirect system for teams that want short links managed as code. The brand should feel trustworthy, calm, precise, and easy to operate.

## Voice

Use direct, operational language.

- Say what the system does.
- Prefer concrete verbs: create, redirect, expire, block, audit, deploy.
- Keep setup and security copy specific.
- Avoid claims that imply tracking, growth, or engagement optimization unless the feature actually supports them.

## Naming

Use `vanityURLs` for the project and `vanityURLs.link` for the public product domain. Use `v8s.link` only when referring to the demo instance or compact badge examples.

## Accessibility

Brand expression must not weaken accessibility. Preserve contrast, visible focus states, meaningful link text, and localized alternatives for badge text.

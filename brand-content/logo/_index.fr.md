Utilisez cette section pour choisir le bon asset de logo vanityURLs et comprendre ce qui manque encore dans le kit de marque.

Gardez les pages Logo centrees sur les standards d'usage, le contexte d'apercu et les tailles d'export requises. Placez les telechargements packages, les bundles ZIP et les inventaires larges d'assets dans [Ressources](/fr/resources/downloads/).

## Groupes de logos actuels

| Groupe                | Usage                                                                            |
| --------------------- | -------------------------------------------------------------------------------- |
| Logo produit          | Identite principale du projet et produit vanityURLs                              |
| Logo d'instance       | Instances de liens courts appartenant a un operateur et pages publiques generees |
| Badges de redirection | Attribution localisee "Redirected by vanityURLs.link"                            |
| Logos du site         | Logos d'entete, favicon, icones d'app et apercus sociaux                         |

## Elements manquants

Pour completer le kit de logos, fournissez les sources en SVG lorsque possible et des exports PNG pour les cibles raster. Les pages enfants listent les tailles et types de fichiers precis.

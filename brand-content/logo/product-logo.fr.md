## Fichiers actuels

| Fichier                               | Format | Usage actuel                    |
| ------------------------------------- | ------ | ------------------------------- |
| `/logo.svg`                           | SVG    | Logo produit general            |
| `/vanityURLs-link.svg`                | SVG    | Marque domaine/produit complete |
| `/vanityURLs-link-logo-slim.svg`      | SVG    | Logo d'entete clair             |
| `/vanityURLs-link-logo-dark-slim.svg` | SVG    | Logo d'entete sombre            |

## A fournir

| Asset                  | Format    | Taille                                      |
| ---------------------- | --------- | ------------------------------------------- |
| Source du logo produit | SVG       | Source vectorielle                          |
| Lockup horizontal      | SVG + PNG | SVG plus PNG a 1200 px de large             |
| Marque produit carree  | SVG + PNG | SVG plus PNG a 1024 x 1024                  |
| Logo monochrome        | SVG       | Variantes claires et sombres en une couleur |

Fournissez les SVG sources d'abord; les exports PNG peuvent etre generes ensuite.

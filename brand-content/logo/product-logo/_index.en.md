## PNG export guidance

Use SVG for product surfaces whenever possible. Export PNG only for targets that cannot consume SVG.

| Use                         | PNG size                                       |
| --------------------------- | ---------------------------------------------- |
| Downloadable lockup preview | 2400 px wide                                   |
| Documentation preview       | 1200 px wide                                   |
| Demo instance logo          | `769 x 501`, matching the supplied SVG viewBox |
| Square avatar or app icon   | Use the website logo set instead               |

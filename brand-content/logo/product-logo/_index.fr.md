## Recommandations d'export PNG

Utilisez SVG pour les surfaces produit lorsque possible. Exportez en PNG seulement pour les cibles qui ne consomment pas SVG.

| Usage                           | Taille PNG                               |
| ------------------------------- | ---------------------------------------- |
| Apercu telechargeable du lockup | 2400 px de large                         |
| Apercu de documentation         | 1200 px de large                         |
| Logo de l'instance demo         | `769 x 501`, comme le viewBox SVG fourni |
| Avatar carre ou icone d'app     | Utiliser plutot le set de logos du site  |

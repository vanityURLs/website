## Recommandations d'export PNG

| Usage                         | Taille PNG                         |
| ----------------------------- | ---------------------------------- |
| Source favicon et icone d'app | `512 x 512` ou plus grand en carre |
| Icone app maskable            | `512 x 512` avec zone sure         |
| Image Open Graph              | `1200 x 630`                       |
| Avatar social carre           | `1024 x 1024`                      |
| Banniere sociale              | `1500 x 500`                       |

`favicon.png`, `favicon-dark.png` et `og-default.png` ont les dimensions de reference de production.

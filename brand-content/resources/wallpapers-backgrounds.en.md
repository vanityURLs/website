## Needed

| Asset | Format | Size |
| --- | --- | --- |
| Desktop wallpaper | PNG or JPG | 3840 x 2160 |
| Desktop wallpaper | PNG or JPG | 2560 x 1440 |
| Mobile wallpaper | PNG or JPG | 1290 x 2796 |
| Presentation background | PNG or JPG | 1920 x 1080 |
| Video-call background | PNG or JPG | 1920 x 1080 |
| Abstract texture source | SVG, PNG, or editable source | Vector or 3840 x 2160 |

Use backgrounds sparingly. They should support brand recognition without reducing legibility.

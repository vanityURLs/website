Contributions are welcome! vanityURLs recognizes [all types of contributions](https://allcontributors.org/docs/en/emoji-key) based on the all-contributors specification.

## Ways to contribute

- **Bug reports** — open an issue in the [GitHub repository](https://github.com/vanityURLs/vanityURLs/issues)
- **Documentation** — improve or translate docs in the [website repository](https://github.com/vanityURLs/website)
- **Code** — submit a [pull request](https://github.com/vanityURLs/vanityURLs/pulls) with fixes or new features
- **Community** — help others in [GitHub Discussions](https://github.com/orgs/vanityURLs/discussions)

This project is released with a [Contributor Code of Conduct](https://github.com/vanityURLs/website/blob/main/.github/CODE_OF_CONDUCT.md). By participating you agree to abide by its terms.

## Pull request guidelines

- Keep PRs focused — one feature or fix per PR
- Update documentation for any user-facing changes
- Add a clear description of what the PR does and why

## Questions?

Start a discussion in [GitHub Discussions](https://github.com/orgs/vanityURLs/discussions) before opening a PR for large changes.

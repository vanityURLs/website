v8s.link is the official vanityURLs demo instance to help you as you build and debug your own instance. You have access to _everything_ related to configuration and operations: the source repository, deployed site, setup values, runtime behavior, live paths, and example link inventory.

The source repository is [vanityURLs/v8s.link](https://github.com/vanityURLs/v8s.link), and the deployed instance is [https://v8s.link](https://v8s.link).

{{< card title="Configuration" icon="file-cog" href="/docs/demo/configuration/" >}}
Inspect the values used by Quickstart, the demo instance settings, and the files that define the operator-owned layer.
{{< /card >}}
{{< card title="Operations" icon="activity" href="/docs/demo/operations/" >}}
Inspect live paths, link inventory examples, lifecycle behavior, and the runtime documentation behind the demo.

Use the demo when your local instance behaves differently from the expected baseline. Start with configuration when setup values or files differ, then move to operations when redirects, status pages, or link states behave unexpectedly.

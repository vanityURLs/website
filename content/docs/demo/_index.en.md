This is the official demo to help you as you build and debug your own instance. Start with [configuration](./configuration/) when setup values or files differ, then move to [operations](./operations/) when redirects, status pages, or link states behave unexpectedly.

The source repository is [vanityURLs/v8s.link](https://github.com/vanityURLs/v8s.link), and the deployed instance is [https://v8s.link](https://v8s.link).

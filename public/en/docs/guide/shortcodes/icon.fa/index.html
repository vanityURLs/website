    آیکون

    آیکون

    برای استفاده از این کد کوتاه درون‌خطی، باید آن را در پیکربندی فعال کنید:

    hugo.yaml
    enableInlineShortcodes: true

    فهرست آیکون‌های موجود را می‌توانید در data/icons.yaml پیدا کنید.

    مثال

    استفاده 

    {{< icon "github" >}}

    آیکون‌های با طرح‌های متفاوت در Heroicons نسخه ۱ خارج از جعبه در دسترس هستند.

    چجوری آیکون‌های خودمان را اضافه کنیم

    پرونده data/icons.yaml را ایجاد کنید، سپس آیکون‌های SVG خود را در فرمت زیر اضافه کنید:

    data/icons.yaml
    your-icon: <svg>your icon svg content</svg>

    سپس می‌توان آن را در کد کوتاه مانند زیر استفاده کرد:

    {{< icon "your-icon" >}}

{{< card icon="your-icon" >}}

    نکته: Iconify Design مکانی عالی برای پیدا کردن آیکون‌های SVG برای سایت شما است.
